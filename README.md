<h1>🔮 Tarot Virtual em Python & SQL 🔮</h1>

## Projeto criado no Jupyter Notebook que realiza o sorteio aleátorio de uma carta no Tarot, tendo como probabilidade exata de 25% de chance** da carta sorteada sair em posição invertida.##


<h2>💻 Tecnologias 💻</h2>
- * Python 3 
- * SQLite (SQL)
- * Jupyter Notebook


<h2>👩‍💻 Na prática 👩‍💻</h2>

1. Clone este repositório na sua máquina.
2. Abra a pasta do projeto no **Visual Studio Code**. 
3. Verifique se tem a extensão **Jupyter** instalada. 
4. Abra o arquivo **`tarot.ipynb`**
5. Clique em **"Run All" (Executar Tudo)** no topo do arquivo ou execute cada célula por vez clicando no botão de **"Play" (Execute)** ao lado delas.
